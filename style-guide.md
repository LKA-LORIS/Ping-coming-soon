# Front-end Style Guide

## Layout

The designs were created to the following widths:

- Mobile: 375px
- Desktop: 1440px

## Colors

### Primary

- Blue: #4F7DF3
- Blue hover : #7195f5

### Secondary

- Pale Blue: #C2D3FF
- Light Red: #FF5263

### Neutral

- Gray: #969696
- Very Dark Blue: #151F29

## Typography

### Body Copy

- Font size: 20px

### Fonts

- Family: [Libre Franklin](https://fonts.google.com/specimen/Libre+Franklin)
- Weights: 300, 600, 700
